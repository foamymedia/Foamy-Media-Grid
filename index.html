the foamy grid

Introduction

The foamy grid is a simple to use responsive CSS grid system, which enables rapid production of mobile and tablet friendly websites without the need for complex frameworks.

The foamy grid was built and designed to be easy to use and also work flawlessly across a wide range of screen sizes and devices.

With breakpoints at 1440px, 1200px, 960px, 768px and 767px the foamy grid covers pretty much every mobile device and also gives a great visual experience whether on a laptop or large screen desktop.

How to use

The foamy grid is very simple to use.

  1. Add a div with class="container" which will hold all of your website.
  2. Add divs with classes such as class="four" ensuring each row adds up to 12 as shown below.
  3. Thats it!

Resize your browser to see the grid below change at regular breakpoints.
